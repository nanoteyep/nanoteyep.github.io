# 숫자 문자열과 영단어
---

## 개요

* 이번 포스팅은 파이썬 코딩테스트 연습문제 풀이입니다.
* `Programmers` 에 공개된 예제이며 링크는 아래와 같습니다.

> <https://programmers.co.kr/learn/courses/30/lessons/81301>
    
---
    
## 코드

```python
def solution(s):
    answer = 0
    voca = {'zero' : '0',
            'one' : '1',
            'two' : '2',
            'three' : '3',
            'four' : '4',
            'five' : '5',
            'six' : '6',
            'seven' : '7',
            'eight' : '8',
            'nine' : '9'}
    
    for key, value in voca.items():
        s = s.replace(key, value)
    answer = int(s)
    return answer
```
