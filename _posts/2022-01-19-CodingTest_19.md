# 네트워크 - 수정중
---

## 개요

* 이번 포스팅은 파이썬 코딩테스트 연습문제 풀이입니다.
* `Programmers` 에 공개된 예제이며 링크는 아래와 같습니다.

> <https://programmers.co.kr/learn/courses/30/lessons/43165>
    
---
    
## 코드

```python
def solution(n, computers):
    rf = []
    for i in range(n):
        temp = ''
        def bfs(i):
            for idx, computer in enumerate(computers):
                nonlocal temp
                if computer[i] == 1:
                    temp += str(i)
                    ir = i
                    while ir < n-1:
                        if computer[ir+1] == 1:
                            bfs(ir+1)
                        ir += 1
        bfs(i)
        if temp != '':
            # rf.append(list(set(temp)))
            rf.append(''.join(sorted(set(temp))))

    temp = []
    for n in rf:
        for m in rf:
            a = 0
            for i in n:
                if i in m and n != m:
                    a += 1
                if a == len(n):
                    temp.append(n)
    for n in set(temp):
        rf.remove(n)
    answer = len(rf)

    return answer
```
